Приложение Calendar обладает следующей логикой:
 
 * при открытии календаря показывается текущий месяц;
 * текущий день выделяется серым цветом;
 * при клике на день он подсвечивается голубым цветом с синей рамкой, появляется форма для добавления/изменения события(localStorage). 
 	Форма имеет следующий функционал:
 	* кнопку "крест"(при нажатии форма исчезает);
 	* четыре окна для ввода информации(есть возможность заполнения не всех форм одновременно,изменения форм при повторном нажатии);
 	* кнопку "готово"(добавляет событие(в localStorage), день с событием выделяется синим цветом);
 	* кнопку "удалить"(удаляет текущее событие(из localStorage));
 * при клике на кнопку "добавить" появляется форма для добавления "быстрого" события.
 	Форма имеет следующий функционал:
 	* кнопку "крест"(при нажатии форма исчезает);
 	* окно для ввода информации; 
 	* кнопку "создать"(добавляет событие(в localStorage), день с событием выделяется синим цветом);
 * при клике на кнопку "обновить" происходит обновление страницы, очистка календаря(удаление всех событий(из localStorage));
 * при клике на кнопку "сегодня" появляется текущий месяц;
 * календарь обладает "плавающей" неделей(появляется и исчезает в зависимости от дня недели первого числа и количество дней в месяце);
 * имеется поиск. Поиск осуществляется только по событиям, которые должны будут произойти после текущей даты. Искать можно по 
   названию события, дате или участникам. При вводе информации появляется окно с возможными вариантами.
